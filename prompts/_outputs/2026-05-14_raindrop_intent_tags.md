# Raindrop Intent Tags — Adopt Going Forward

These are tags Lee adds **by hand** on save (or during a sweep pass). Auto-tagger won't produce them because they encode intent, not content. Five to seven is plenty — they're discriminators, not categories.

## Proposed set

| Tag | Use it when… | Example signal |
|---|---|---|
| `re-read` | You've read this once and felt it pay rewards on the re-read, or you know you'd want to come back to it. | Hamming, Mills, Sennett, anything that's been on the shelf-of-mind for a year. |
| `for-stacks` | Could earn a Stacks slot eventually. Lower bar than re-read; just "worth promoting." | Strong essay/site you'd point a colleague at. |
| `for-piece` | Feeding active writing. Stack per-piece (e.g., `for-ic-p2`, `for-consulting-page`) if you want fine-grain. | Citation candidates while drafting. |
| `craft-fuel` | The craft / maintenance / material-culture lineage. Mills, Sennett, van der Ryn, dispatchesmag, Lost Art Press. | When you save another piece on woodworking-as-thinking, attention-as-craft, etc. |
| `consulting-fuel` | The Visa/Rao/Schein operational-honesty line. | Coordination posts, consulting-craft essays, "how I work" pieces. |
| `discard` | Saw it, decided no. Alternative to deletion — preserves search trace ("I considered this once"). | Use sparingly; deletion is also fine. |

## Optional add-ons

- `quote-bait` — saves with one or two great pull-quotes you'd cite elsewhere.
- `gift` — bookmarks you'd send to a specific person (could be `gift:lee`, `gift:ashley`, etc.).
- `pending-read` — saved but not yet read; clears off when you read it.

## How to use

- **Save → tag-then-edit pattern**: let Raindrop auto-tag, then while you're in the save panel, add one intent tag if applicable. ~5 seconds extra per save.
- **Weekly sweep**: 10-minute pass through Unsorted, eyeball any save that triggers a re-read/for-stacks/craft-fuel reaction, hand-tag accordingly. Catches the intent the auto-tagger can't.
- **At write-time**: when starting a piece, search `topic:Design` + `craft-fuel` to pull just the craft-grade design saves. Composable filters.

## Don't bother with

- `important` / `priority` / `starred` — Raindrop already has an "important" flag built-in; use it.
- `read` / `unread` — Raindrop has built-in read state.
- `interesting` — too vague to filter on. Everything saved was interesting.
